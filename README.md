## Development

Download:    
```shell
git clone git@github.com:alexander-shvets/writebar.git
cd writebar
yarn
```

Install dev Dependencies:    
```shell
npm install -g electron
npm install -g electron-packager
npm install -g electron-installer-dmg
```

Run in dev mode:    
```shell
yarn start
```

Build App and Installer:    
```shell
yarn && yarn build
```
